# Introduction

JSON API under /api (auth email OTP, CRM compatibility layer). Admin features live in the web app.

<aside>
    <strong>Base URL</strong>: <code>http://127.0.0.1:8000</code>
</aside>

    JSON routes are mounted under <code>/api</code> (see the base URL above). Many admin features are served from the Inertia web app under <code>/admin</code> instead of this API.

    <aside>Use the sidebar links for the <strong>Postman collection</strong> (v2.1 JSON) and the <strong>OpenAPI</strong> YAML spec. After changing routes or Scribe docblocks, run <code>php artisan scribe:generate</code> or <code>composer docs</code>.</aside>
